## Verified HDCP Math (confirmed with Josh)
- **9-hole HDCP:** `round(full_HDCP / 2)`
- **Stroke dots:** `round((higher_9h - lower_9h) / 2)` — placed at hardest SI holes first
- **Hole↔SI mapping:** H1=4, H2=7, H3=2, H4=6, H5=8, H6=5, H7=9, H8=1, H9=3
- **Hardest holes (by SI):** H8(1), H3(2), H9(3), H1(4), H6(5), H4(6), H2(7), H5(8), H7(9)
- **Matchup pairing:** Players 1&2 = Matchup 1, Players 3&4 = Matchup 2
  - Higher HDCP player in each pair gets the stroke dots

## CSS Architecture
- `.card`: 8.5in × 5.4in, `border-bottom: 2px solid #d4af37`
- `.cards-wrapper`: flex column, no page-break rules in print (let browser handle 2-up)
- `@media print`: removes color, sets B&W, `@page { size: 8.5in 11in; margin: 0; }`
- Uniform cells: all `td` use `table-layout: fixed`, explicit `col` widths
  - Name: 54pt, Holes: 32pt each, Out: 33pt, HDCP: 33pt, Pts: 40pt
- `.matchup-divider`: 3pt height, solid dark line
